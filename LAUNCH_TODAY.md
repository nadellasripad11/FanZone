## 🔐 Important Before You Launch

### Firebase Setup (1 minute)
1. Go to: https://console.firebase.google.com/project/fanzone-worldcup
2. Realtime Database → Rules tab
3. Replace all rules with contents from `DATABASE_RULES.json`
4. Click "Publish"

**Without this step, reactions won't save correctly.**

### Create Test Live Match (30 seconds)
1. Realtime Database → Data tab
2. Click "+" next to "fanzone-worldcup"
3. Name: `live_match`
4. Add this data:
```json
{
  "home": "🇧🇷 Brazil",
  "away": "🇦🇷 Argentina",
  "score_home": 2,
  "score_away": 1,
  "minute": 78
}
```

**Without this, live match info won't display.**

## 💡 Optimization Tips for Scale

### If reactions are slow (>1 second):
- Clear browser cache (Cmd/Ctrl + Shift + R)
- Close other tabs
- Check your internet speed

### If opponent won't connect:
- Both must be signed in with Google/Email (not guest)
- Both must be in penalty game at same time
- Try refreshing both browsers

### If penalty game glitches:
- Click "Close" button
- Wait 2 seconds
- Click "Penalty" tab again
- Try "vs AI" mode (more stable)
